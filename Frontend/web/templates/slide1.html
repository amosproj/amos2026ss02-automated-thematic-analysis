Group 2: Automated Thematic Analysis

What is Thematic Analysis?

  • Researchers conduct interviews and collect transcripts.
  • A theme is a recurring pattern across interviews.
  • A codebook catalogs all discovered themes with definitions.
  • Coding tags every relevant quote with its theme.
  • Done manually, this takes weeks for a single study.
